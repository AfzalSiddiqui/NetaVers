# NetaVers
React Native (Expo) app for browsing **multiple verses** (list + detail), with search and local favorites.

## Run

1) Install dependencies

```bash
npm install
```

2) Start the app

```bash
npm run start
```

Then scan the QR code with **Expo Go** (iOS/Android) or run on an emulator.
